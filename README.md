# translate-with-gpt-denops

- Neovim 内でテキストを選択して`:Translate`

- ※ OpenAI の API キーが必要です

- ※ OpenAI の API キーを使うと別途料金がかかります．

## install

- dein

```vimscript
call dein#add("tai-no-ala-ni/translate-with-gpt-denops")
```

## requirements

- Neovim v0.10.0
